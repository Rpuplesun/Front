input type

file : 원하는 파일을 선택하여 서버로 전송하는 입력 상자
email : 이메일을 입력받는 글 상자 @가 없으면 제출되지 않음
url : 웹사이트 입력받는  글 상자 (http로 시작하는 url)
tel : 전화 번호 입력난 글 상자

input 속성
size : 박스의 크기를 설정하는 속성
maxlength : 값의 최대 길이를 설정하는 속성
checked: 미리 선택되어있는 옵션을 설정하는 속성 (checkbox, radio)

form 태그

웹 페이지에서 사용자로부터 입력을 받아 처리쪽으로 보내줄 때 사용하는 태그
사용자가 입력한 데이터를 서버쪽으로 보낼 때 form 요소 사용

<form action="/login" method ="GET(POST)">

    <p>아이디: <input type="text"></p>
    <p>암호 <input type="password"></p>
    <button type="submit">로그인</button>

</form>

method 방식
    get = url에 데이터를 저장하여 전송하는 방식
    post = body에 데이터를 저장하여 전송하는 방식

body ---> 통째로
query -->code="31234"
param ---> /312345/

table

<table border ="테두리(숫자)", width="가로길이(%, px)", align="정렬(center,right, left)">
    <tr> ---- 각 테이블의 층(row)
        <td width="길이", heigth="높이", rowspan="세로병합", colspan="가로병합"></td> ... 데이터
        <th></th> ... 제목
    </tr>
</table>
